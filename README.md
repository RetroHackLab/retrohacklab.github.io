# 📱 Retro HackLab's Repo

Welcome to **Retro HackLab's Repo**! This platform serves as a specialized archive and preservation hub for legacy iOS modding tools, cross-platform terminal utilities, and custom device exploits.

## 🌍 Ecosystem Links
* 📥 **Cydia Source URL:** `https://github.io` *(Add directly inside your iOS package manager)*
* 🖥️ **Google Sites Interface:** [https://sites.google.com/view/retrohacklab/repository](https://sites.google.com/view/retrohacklab/repository) *(Contains web-based `.deb` downloads and installation guides)*

---

## 🛠️ Integrated Toolkit
* 🔄 **dos2unix**: Line-ending conversion utility (CRLF to LF) ensuring script compatibility between Windows and Unix/Linux environments.
* 💾 **Legacy-iOS-Kit**: Comprehensive suite for legacy 32-bit/64-bit device downgrades, SSH Ramdisk generation, and custom SSH bypasses.
* 🔌 **Checkm8-a5**: Patched hardware exploit Arduino `.ino` sketch optimized for stable execution on A5 silicon.
* ❄️ **Sn0wbreeze (iH8sn0w)**: Archival custom IPSW creation engine built for legacy ARM and A4 devices.

---

## 📥 Package Manager Setup
1. Launch **Cydia**, **Sileo**, or **Zebra** on your jailbroken iOS device.
2. Navigate to the **Sources** tab.
3. Select **Edit** then **Add** in the upper right.
4. Input the official URL: `https://github.io`
5. Refresh your sources to access the hosted packages.
